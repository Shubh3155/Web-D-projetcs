# Projects Workspace

- tailwind-landing/ — Tailwind CDN landing page
- portfolio-tw/ — Tailwind CDN portfolio
- tailwind-store/ — Tailwind CDN product grid
- react-dashboard/ — Vite + React + Tailwind app with routing and components
- next-landing/ — Next.js + Tailwind landing

See each project's README for run instructions.
