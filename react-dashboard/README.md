# React Dashboard

Dev

```bash
npm install
npm run dev
```

Build

```bash
npm run build
npm run preview
```

This project uses Vite + React + TailwindCSS. Tailwind is configured in `tailwind.config.cjs` and PostCSS in `postcss.config.cjs`. Styles are in `src/styles.css`. Starter routes/components are in `src/`.
